# SheikhApp - مدينة أولاد الشيخ (متجر جملة)

تطبيق تسوق إلكتروني متطور لبيع المنتجات بالجملة، مصمم خصيصاً للتجار والعملاء في السوق المصري.

## 🏪 أقسام المتجر

- **مياه** - مياه معدنية ومياه شرب
- **مشروبات** - مشروبات غازية وعصائر
- **جبن وألبان** - منتجات الألبان والجبن
- **ورقيات** - أدوات مكتبية وورقية
- **حفظات** - أدوات حفظ وتخزين
- **ماكولات أساسية** - الزيوت والسكر والأرز
- **منظفات ومعطرات** - مواد تنظيف ومعطرات
- **بقالة** - مواد غذائية متنوعة
- **عناية شخصية** - مستحضرات تجميل ونظافة
- **مستهلكات وبطاريات** - بطاريات ومستهلكات كهربائية
- **صابون** - صابون ومنظفات
- **تبريد وتجميد** - ثلاجات ومجمدات
- **ياميش رمضان** - مستلزمات رمضان
- **شراب وصوص** - شراب وصلصات
- **حلويات** - حلويات ومعجنات

## 🚀 الميزات الرئيسية

### 🛒 نظام البيع بالجملة

- **وحدات قياس مرنة**: كيلو، قطعة، علبة، صندوق
- **إدخال كمية مخصصة**: يمكن للعميل إدخال الكمية التي يريدها مباشرة.
- **تحقق من المخزون**: يظهر تنبيه إذا كانت الكمية المطلوبة أكبر من المتاح.
- **عرض واضح**: إظهار الوحدة والكمية في السلة والطلبات

### 👨‍💼 لوحة الإدارة

- إضافة وتعديل وحذف المنتجات والأقسام
- تحديد وحدة القياس لكل منتج
- تخصيص الكميات المتاحة للبيع
- **إدارة المخزون**: تحديد الكمية المتاحة لكل منتج.
- **الحد الأدنى للطلب**: تحديد أقل كمية يمكن للعميل طلبها من كل منتج.
- متابعة الطلبات وإدارتها

### 📱 تجربة مستخدم متطورة

- واجهة عربية كاملة مع دعم RTL
- تصميم متجاوب للهواتف والتابلت
- سلة مشتريات ذكية
- نظام مصادقة آمن
- تتبع الطلبات

### 🔄 تحديثات تلقائية

- البيانات تتحدث فوراً عند التعديل من لوحة الإدارة
- لا حاجة لإعادة تحميل الصفحة
- متزامن مع قاعدة البيانات

## 🛠️ التقنيات المستخدمة

- **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
- **Backend**: Firebase (Firestore, Authentication)
- **Icons**: Lucide Icons
- **Fonts**: Google Fonts (Cairo)

## 📦 التثبيت والتشغيل

1. قم بتحميل المشروع:

```bash
git clone https://github.com/yourusername/SheikhApp.git
cd SheikhApp
```

2. شغل خادماً محلياً:

```bash
python -m http.server 5500
```

3. افتح المتصفح واذهب إلى: `http://localhost:5500`

## 🔧 إعداد Firebase

1. أنشئ مشروعاً جديداً في [Firebase Console](https://console.firebase.google.com/)
2. فعّل Firestore Database
3. فعّل Authentication مع Anonymous sign-in
4. انسخ إعدادات المشروع إلى `js/app.js`

## 🎯 كيفية الاستخدام

### للعملاء:

1. تصفح المنتجات حسب الأقسام
2. أدخل الكمية المطلوبة مباشرة في حقل الإدخال
3. أضف للسلة وأكمل الطلب

### للمديرين:

1. سجل دخولك كمدير
2. اذهب لقسم "الحساب" وستجد لوحة التحكم
3. أضف منتجات جديدة مع تحديد الوحدات، الكمية المتاحة في المخزون، والحد الأدنى للطلب.
4. راقب الطلبات الواردة

### مثال على الاستخدام:

**للمدير:**

1. أضف منتج "زيت عباد الشمس"
2. حدد وحدة القياس: "كيلو"
3. حدد الكمية المتاحة في المخزون: "500" كيلو
4. حدد الحد الأدنى للطلب: "5" كيلو
5. حدد السعر: 25 ج.م للكيلو

**للتاجر:**

- يدخل الكمية المطلوبة، مثلاً "20" كيلو. إذا أدخل "3" كيلو، سيظهر تنبيه بأن الحد الأدنى هو 5 كيلو. إذا أدخل "600" كيلو، سيظهر تنبيه بأن المتاح هو 500 كيلو.
